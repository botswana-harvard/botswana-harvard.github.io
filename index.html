The EDC

A data management system for audited public health research

An open data collection and management system designed to be customized. Clinical trials have a lot in common but not enough to be data managed by just one system.

  • written in python with a lot of Django
  • core modules - edc-base, edc-consent, edc-visit-tracking, edc-visit-schedule, edc-rule-groups, edc-locator, edc-requisition, etc
  • optional modules - edc-sync, edc-call-manager, etc

Open-source project management

  • git-flow managed releases and hotfixes
  • openly publish all code and all commits
  • semantic versioning
